## Places to Visit in India - Data Analysis
- A data analysis project involving a dataset 'Top Indian Places to Visit.csv' attached in this repository, attached in the /data sub-folder.
- The '/notebooks/exploratory_analysis.ipynb' file contains the basic data analysis and statistical information, while the '/notebooks/visualizations.ipynb' file contains visual representations of data using bar graphs, heatmap, scatter plot and box plot.

## Tech Stack
- Python (NumPy, Pandas, Matplotlib, Seaborn)
- VS Code / Jupyter Lab / Google Colab
- Git and GitHub
